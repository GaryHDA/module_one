Finding what you need

The search is done by URL structure and not search terms.

In the beach litter section the smallest unit is one aggregated observation of one item type. For example:


{'location':a-beach, 'date':a-day, 'length':beach-length-m, 'quantity': # pieces, 'code':MLW-code, 'project':a-project}

The code is the basic unit of counting and then it can be grouped or sorted by location.

So you would start with the key-word 'codes' and then choose a location


~/dirt/codes/Vevey: will give you all the code data for the city of Vevey:


http://mwshovel.pythonanywhere.com/dirt/codes/Vevey/

Suppose you only want data on plastic films in Cordova Creek?


http://mwshovel.pythonanywhere.com/dirt/codes/Cordova-Creek/G67/

Where 'G67' is the code for plastic films

If you get it wrong, no worries. It will throw a 404 error and on that page their will be links to help you get what you want

Getting beach information is the same, the key word is 'beaches':


http://mwshovel.pythonanywhere.com/dirt/beaches/Lac-L%C3%A9man/

Will give you a set of data that has the geo data for all locations on 'Lac LÃ©man'.

Aggregated Data

Their are two levels/types of aggregation

-->Summary data

-->Daily totals

Sumary data key word 'summary':

  • Number of locations
  • Total pieces
  • Num of samples
  • First sample
  • Last sample
  • Average pieces per meter
  • Max pieces per meter
  • Min pieces per meter
  • 25th percentile
  • 75th percentile
  • Number of lakes
  • Number of rivers
  • Standard deviation
  • Search term

So the following would give you the summary data for all the surveys on the Aare river

http://mwshovel.pythonanywhere.com/dirt/summary/Aare/

Daily totals key word 'daily-total:

This adds up all the code values for a given day and groups them by location. If you search for a river then you would get all the daily totals for all the surveys on that river

{"location": a-beach, "date": a-date, "length": length of beach, "total": pieces}

http://mwshovel.pythonanywhere.com/dirt/daily-total/Aare/

Returns all the daily totals for the Aare.
