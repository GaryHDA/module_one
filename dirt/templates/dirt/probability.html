Project :The binomial variable
Recurring events

Alot of emphasis is given to calculating "How much trash?" is in the environment. I don't think it is possible to calculate any number that has any real meaning. However we can calculate the probability of how much trash you may find each time you go to the shore. The differnce is subtle, but we can test the results of a probability model far more reliably than an extraplotation.

Testing the probability model
To test the probability model, all you have to do is go down to the beach and do a litter-inventory. The bonus is that the results from that inventory will be added to the model. Thus refining the model each time you go out.
Inspiration and motivations: 26-09-2018
The binomial variable has come up several times while I was looking for a way to express the siginifigance of results or predict a value:

  • Determining if the microbio results were just random
  • Predicting the qty of trash you might find per meter of shoreline
Binomial succes or not
The questions both ask the same thing "What are the chances that 'this event' will happen ?". In the first case if the probability is high then it would not be prudent to associate other independent variables with the 'event'. In the second case if the probaility is high we may be approaching a baseline value for deteriming trash levels on the shoreline.
